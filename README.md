## Crypto Hub

### Description

CryptoHub is a web3 aggregation tool that supports multi-chain queries, including ERC20, Native Tokens, NFTs, and DeFi.

### Features

- Multi-chain queries
- ERC20, Native Tokens, NFTs, and DeFi
- Batch query
- more features in coming...

### Tech Stack

- Next.js
- Tailwind CSS
- TypeScript
